# raspberry4
收集一些可用的raspberry4的脚本小程序
- buzzer.py   蜂鸣器
- dht11.py    温湿度
- redbuzzer.py  红外报警
- deng.py 4位共阴数码管
- MQ-2.py 烟雾报警
- JQC-3FF.py 继电器
- lightdz.py 光敏电阻/声敏电阻 控制LED
